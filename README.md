# Online_Alot

## Landing Page Navigation

Follow these steps to navigate the landing page (`activity_main.xml`):

1. Open the app and wait for the landing screen to load.
2. You will see the brand title (`Cutify`) and hero text on top of the background image.
3. Scroll is not required on this screen; the main action is the **Continue** button at the bottom center.
4. Tap **Continue** (`btn_continue_user`) to proceed to the next screen.
5. If the button does not appear centered on your device, update to the latest build where the button is forced to center alignment.

## Notes

- Landing page layout file: `app/src/main/res/layout/activity_main.xml`
- Continue button id: `btn_continue_user`
